# 🔐 File Packer-Unpacker (C-Based Implementation)

A system-level project developed using **C programming** that demonstrates file handling, data packing, and unpacking concepts.  
This project simulates how multiple files can be combined into a single file and later extracted back while preserving their content.

---

## 📦 Features

- 🗃️ Pack multiple files into a single file  
- 📂 Unpack files back into original format  
- 📄 Supports multiple file types (text, code, etc.)  
- ⚙️ Uses low-level file handling operations  
- 🧪 Includes multiple test files and programs  
- 🏁 Executable (`.exe`) available for direct execution

## ⚙️ How It Works

### 🧳 Packing Concept

- Reads data from multiple files  
- Combines content into a single file  
- Maintains structure for later extraction  

### 📤 Unpacking Concept

- Reads the combined file  
- Extracts stored data  
- Recreates original file contents
